# Chat on Chell

## About

`Socket.io`を使ってシェル上でチャットができます。

## Environment

- Node.js
  - Socket.io-client

## Usage

下記のコマンドを実行します。

```sh
# install packages
$ npm install

# execute application
$ node app.js
```

実行すると、名前を聞かれるので入力し、
続いてメッセージを入力することでチャットができます。

```sh
user name: user1

hi

user1: hi

user2: hi
```
